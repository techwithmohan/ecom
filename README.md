# Urban Threads E-commerce Website

A modern e-commerce platform built with Next.js and Tailwind CSS.

## 🚀 Features
- **Local Storage Mode**: No database setup required for the demo.
- **Admin Panel**: Manage products, customers, and orders.
- **Responsive Design**: Mobile-first approach.
- **Cart System**: Fully functional shopping cart.

## 🛠️ Deployment
This project is ready for deployment on **Vercel** or **Netlify**.
See [how to deploy.md](how%20to%20deploy.md) for detailed instructions.
